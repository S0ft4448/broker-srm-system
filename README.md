# 📊 Broker SRM System

**Broker SRM System** — это специализированная CRM/SRM-система, разработанная для брокеров и инвесторов. Главная цель проекта — максимально ускорить обработку входящих заявок, автоматизировать рутинные действия и сделать процесс взаимодействия между всеми участниками сделки простым и прозрачным.

---

## 🚀 Основная цель проекта
Облегчить и оптимизировать рабочий процесс для финансовых брокеров и инвесторов за счет автоматизации обработки заявок (запросов на инвестиции, кредиты, подбор объектов и т.д.). Система сводит к минимуму человеческий фактор и бумажную волокиту, позволяя закрывать сделки быстрее.

## ✨ Ключевые возможности (Фичи)
* **Быстрая обработка заявок:** Удобный интерфейс для мгновенного приема, фильтрации и распределения входящих запросов.
* **Кабинет инвестора и брокера:** Разделение прав доступа и уникальный функционал для каждой роли.
* **Трекинг статусов:** Наглядное отслеживание того, на каком этапе находится каждая заявка (в обработке, одобрено, на согласовании, успешно завершено).
* **Уведомления:** Оперативное информирование о новых заявках или изменении их статуса.

# 📊 Broker SRM System

**Broker SRM System** is a specialized CRM/SRM platform designed for brokers and investors. The main goal of the project is to significantly accelerate application processing, automate routine tasks, and make the interaction between all parties involved simple and transparent.

---

## 🚀 Main Goal
To streamline and optimize the workflow for financial brokers and investors by automating the processing of applications (requests for investments, loans, property selection, etc.). The system minimizes human error and paperwork, allowing deals to be closed much faster.

## ✨ Key Features
* **Fast Application Processing:** An intuitive interface for instant intake, filtering, and distribution of incoming requests.
* **Broker & Investor Dashboards:** Role-based access control with unique functionality tailored for each user type.
* **Status Tracking:** Visual monitoring of the stage each application is in (e.g., In Progress, Approved, Under Review, Successfully Completed).
* **Notifications:** Real-time alerts for new applications or status updates.
